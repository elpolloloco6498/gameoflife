Game of Life Programmed in Javascript

Conway's Game of Life in Canvas

Cellular automaton

The game of life is not a game is a cellular automaton. The program evolves with pre established rules that gides the evolution of the system. The game of life is a very simple model of life. The black squares represents an individual, let's take a bacteria for example. If there are too many organisms around the black square, it dies. If there are not enough it dies. If there are exactly 3 organism around a blank square, an organism is born.

The rules are really simple, and we can notice that some configurations are stable. For more informations on Conway's game of life visit GAME OF LIFE WIKIPEDIA and also the conway Wiki page LIFE WIKI
